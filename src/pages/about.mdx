# About

My name is Alice (Xinyun). I'm a software engineer based in Melbourne, Australia.

I write code for a living, take photos of quiet moments, build tiny keyboards, and learn French badly but with great enthusiasm. I have a background in software engineering and data science, and I currently work on developer tools at Zendesk.

This site is my digital garden — a place where I think out loud, connect ideas, and write things down so I remember them. Nothing here is polished. That's the point.
